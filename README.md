# IPL Data Analysis using Python

This project analyzes IPL match and ball-by-ball data using:

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analyses

- Top Run Scorers
- Most Sixes
- Most Fours
- Top Wicket Takers
- Toss Decision Trends
- Toss Impact on Match Results
- Matches Per Season
- Top Venues
- Winning Methods Analysis

## Skills Demonstrated

- Data Cleaning
- Data Exploration
- Data Aggregation
- Data Visualization
- Insight Generation
